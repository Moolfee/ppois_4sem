# CLI-команды

## Формат команд

- Команда: `<group> <action> <args...>` или одиночная (`play`, `help`).
- Значения с пробелами передаются в кавычках.

Пример:

```text
video add "My Demo Video" mp4 120
```

## Базовые команды

### `help`

Показывает справку и список поддерживаемых форматов.

### `exit`

Завершает приложение.

### `status`

Показывает текущее состояние:

- выбранное видео,
- статус воспроизведения,
- громкость,
- яркость,
- размер библиотеки,
- число плейлистов.

## Управление воспроизведением

### `play`

Запускает воспроизведение выбранного видео.

### `pause`

Ставит воспроизведение на паузу.

### `stop`

Останавливает воспроизведение.

## Поддерживаемые форматы

### `formats list`

Показывает текущий набор поддерживаемых расширений.

## Блок `video`

### `video add <title> <format> <duration_seconds>`

Добавляет видео в библиотеку.

Пример:

```text
video add "Lecture 1" mp4 3600
```

### `video list`

Выводит список всех видео в библиотеке.

### `video select <title>`

Выбирает видео как текущее.

### `video remove <title>`

Удаляет видео из библиотеки.

Побочные эффекты:

- удаляет видео из всех плейлистов;
- если удалено текущее видео, воспроизведение останавливается.

## Блок `playlist`

### `playlist create <name>`

Создает плейлист.

### `playlist list`

Показывает список плейлистов.

### `playlist add <name> <video_title>`

Добавляет видео в плейлист.

### `playlist remove <name> <video_title>`

Удаляет видео из плейлиста.

### `playlist show <name>`

Выводит содержимое плейлиста.

### `playlist select <name> <video_title>`

Выбирает видео из плейлиста как текущее.

## Настройки

### `volume set <0-100>`

Устанавливает громкость.

### `brightness set <0-100>`

Устанавливает яркость.

## Типовые ошибки пользователя

- Неверное количество аргументов -> вывод usage.
- Неподдерживаемый формат -> `UnsupportedFormatError`.
- Не найдено видео/плейлист -> `VideoNotFoundError` / `PlaylistNotFoundError`.
- Некорректное числовое значение -> сообщение `must be an integer`.
